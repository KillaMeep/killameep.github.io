ASCII-gen

ASCII-gen is an open-source tool that transforms images and video frames into ASCII art, offering users an engaging way to convert visual content into text-based representations. The system processes image data efficiently, generating detailed ASCII art by analyzing pixel patterns and mapping them to characters. It supports both static images and videos, converting frames into sequences of ASCII art and exporting optimized GIFs. Written in Python, this project is fully compatible with both Linux and Windows environments, providing developers and creatives with a versatile tool for artistic and practical applications.

StarNodeAnimator

StarNodeAnimator is an open-source software that generates stunning star animations using OpenGL within Pygame. The program creates randomly placed stars with customizable sizes and connects them based on user-defined parameters. These animations are rendered into frames and compiled into MP4 videos, with options for configuring colors, node generation, and screen dimensions. Offering both a graphical user interface and a command-line interface, StarNodeAnimator allows for flexible interaction, making it ideal for generating captivating visual content with ease.

BeatBlock AI

BeatBlock AI is an advanced real-time object detection system designed for gaming environments. At its core is the predict script, which utilizes the YOLOv5 framework to identify and classify in-game elements like Blocks, Inverse, Pipes, Sliders, Players, and Bombs directly from the game screen. The system captures screen content using mss and processes it with torch for model inference and OpenCV (cv2) for image handling, ensuring high performance with minimal latency. A well-defined data.yaml file sets precise training and validation parameters, making BeatBlock AI a powerful tool for enhancing gameplay and enabling game analysis.
